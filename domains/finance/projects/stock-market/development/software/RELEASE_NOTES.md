# Stock Prediction — Release Notes

## v1.0.0 (Initial Release)

### Features
- **656-Parameter Interactive Prediction**: Set weights and directions for 656 financial factors across 6 domains (Macro & Monetary, Fundamental, Cross-Asset, Technical, Sentiment, Geopolitical)
- **On-Device LLM Research Assistant**: Download Phi-3.5 Mini, Gemma 2, or Llama 3.2 for private, offline AI-powered parameter research
- **Automated ONNX Prediction**: GradientBoosting model runs entirely in-browser/on-device for automated direction forecast
- **Parameter Definitions**: Every parameter has a plain-English and technical description
- **Snapshot History**: Save, export, and review prediction snapshots in JSON, CSV, PDF, or Excel format
- **Cross-Platform**: Available on Web, iOS, Android, and Desktop

### Privacy
All predictions and AI inference are computed on-device. No personal data is transmitted or stored externally.

### Platforms
- Web: https://bpupadhyaya.github.io/prediction/stock-prediction/
- iOS: Requires iOS 17.0+, Apple Silicon recommended for LLM features
- Android: Requires Android 8.0+ (API 26), 8+ GB RAM recommended for LLM features
- Desktop: Requires Python 3.11+, run locally
